#  Exam Score Prediction System

##  About
This is a Machine Learning project that predicts a student's exam score based on study-related inputs. The application is built using Python and Streamlit.

##  Features
- Predict exam scores
- Simple and interactive UI
- Machine Learning based prediction

##  Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn

## ▶️Run the Project

streamlit run stream.py
